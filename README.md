# Sift

**Discover music you've never heard — and will love.**

Sift is a music discovery app that learns your taste and surfaces new songs from an
artist-relationship graph. It has a fast swipe interface, 30-second previews that jump to
the *best part* of each track, and an explicit **love / not-for-me / already-heard**
feedback loop that sharpens recommendations as you go.

## Try it

Open **`index.html`** in a browser, pick a few genres/artists (tap a genre up to 3× to
weight it), and start swiping.

## How it works

- **Discovery graph + previews** via the Deezer open API (related artists → their tracks)
- **"Best part" auto-jump** via Web Audio energy analysis (lands on the hook/drop/chorus)
- **Taste + finds persist locally** — no account needed

*Early prototype — built as a phone-feeling web app before going native for iOS.*
